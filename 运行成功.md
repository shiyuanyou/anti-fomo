## 如何使用

### 方式一：演示模式（推荐先尝试）

使用模拟数据快速体验系统功能：

```bash
conda activate finance
python demo.py
```

### 方式二：真实数据模式

在网络连接正常时，使用真实市场数据：

```bash
conda activate finance
python run.py
```

## 配置您的组合

编辑 `config.yaml`：

```yaml
portfolio:
  holdings:
    # 修改为您的实际持仓
    - symbol: "000510"
      name: "中证A500"
      type: "index"
      allocation_type: "amount"  # 或 "ratio"
      value: 50000  # 金额或百分比
```

## 调整阈值

根据您的风险偏好调整：

```yaml
thresholds:
  portfolio_volatility:
    warning: 3.0   # 更敏感: 降低此值
    alert: 5.0     # 更宽松: 提高此值
```

## 启用 AI 分析（可选）

在 `config.yaml` 中配置：

```yaml
ai_analysis:
  enabled: true
  api_key: "sk-your-openai-api-key"
```
