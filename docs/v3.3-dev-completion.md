## 主要API端点

### Templates（官方模板）
- `GET /api/templates` - 获取所有模板
- `GET /api/templates/{template_id}` - 获取单个模板

### Shares（用户分享）
- `POST /api/shares` - 创建新的分享配置
- `GET /api/shares/{share_id}` - 获取分享配置

### Assets（本地模式）
- `GET /api/assets` - 读取config.asset.yaml
- `POST /api/save` - 保存到config.asset.yaml

## 架构说明

**当前架构：**
```
Frontend (Vue 3) → Nginx → FastAPI (8000) → SQLite (anti-fomo.db)
                          ↓
                    config.asset.yaml (Local Mode)
```

**运行模式：**
- **本地开发**：前端 `npm run dev` + 后端 `python3 -m uvicorn main:app --reload --port 8000`
- **生产部署**：Nginx托管静态文件 + FastAPI服务
