# 二、Anti-FOMO v1 MVP 功能范围（只做这些）

## MVP必须实现（P0）

### ① 市场估值引擎（核心）

使用 akshare：

```
沪深300
中证500
上证50
```

输出：

```
PE百分位
PB百分位
市场区间
```

### ② 市场状态评分（最关键模块）

统一输出：

```
市场状态 = {过热 / 正常 / 便宜}
```

简单规则：

```
score = (PE_percentile + PB_percentile) / 2
```

```
> 80 → 过热
20–80 → 正常
< 20 → 便宜
```

### ③ 决策节奏控制器（Anti-FOMO核心）

输入：

```
个人波动
市场估值
```

输出：

```
需要看盘？
是否忽略市场？
是否计划再平衡？
```

这是产品差异点。

### ④ 周度市场报告（自动生成）

输出：

```
市场估值变化
资产状态变化
行为建议
```

只做文本。

# 五、核心算法设计（Anti-FOMO的灵魂）

这是产品最有价值部分。

## 决策逻辑（建议）

### 是否需要看盘

```
if 资产波动 > 阈值 AND 市场极端:
看盘
else:
忽略
```

### 是否需要调整

```
if 市场便宜 AND 现金多:
建议买入

if 市场过热 AND 权重过高:
建议减仓
```

### 是否需要再平衡

```
权重偏离 > 5%
```
