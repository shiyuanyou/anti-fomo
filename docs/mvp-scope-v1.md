# anti-fomo 官网简洁版 MVP 范围冻结（V1）

## 1. 文档目的

本文件用于冻结 V1（官网简洁版）范围，保证按时上线并避免需求发散。

适用版本：V1（云端免登录版本）

---

## 2. V1 核心定位

- 用户无需登录即可上手
- 在 3 分钟内完成首次核心体验
- 传达产品价值：资产配置认知与反 FOMO 决策辅助

---

## 3. 必做范围（In Scope）

### 3.1 Home Page

必须包含：

1. 三个基础功能截图 + 简述
2. 项目初衷（为什么做 anti-fomo）
3. 本地部署扩展能力介绍
4. 社区介绍入口
5. 维护者信息
   - GitHub 链接
   - 联系方式（微信小号、邮箱）

### 3.2 指数展示页（基础版）

每个指数条目需支持：

1. 名称、代码
2. 指数价值曲线（可选叠加全收益曲线，若数据可用）
3. 基础统计（优先级从高到低）：
   - 平均年化收益
   - 周度最大回撤 / 最大增长
   - 年度最大回撤 / 最大增长

查询筛选（Web 版必做）：

1. 指数名模糊查询
2. 指数代码查询
3. 按基础信息过滤（类型/区域/风格）

### 3.3 模板对比功能（基础版）

V1 只做：

- 组合价值信息摘要
- 全收益信息摘要（若可用）

V1 暂不做：

- 多模板复杂摘要一键复制
- AI 对话增强

### 3.4 社区页（最小版本）

只做静态展示：

- 社区定位
- 未来可参与方式

---

## 4. 明确不做（Out of Scope）

以下内容全部放入 V2/V3：

1. AI 查询、AI 对话、AI 策略拟合
2. 购物车复杂能力与 AI 分析入口
3. 组合包分享/下载闭环
4. 插件系统
5. 订阅模式与 token 共享能力
6. 大量宏观历史数据扩展专题（保险、存款、通胀、海外同类）

---

## 5. 数据与 API 最小要求

数据层：

- 至少覆盖核心主流指数元数据：名称、代码、类型、区域、风格
- 支持周/年两个基础统计尺度

API 层（最小）：

- 指数列表查询（含筛选参数）
- 单指数基础统计查询
- 模板对比摘要查询

---

## 6. 质量与验收标准

### 6.1 体验验收

- 首屏加载可用
- 用户 3 分钟内完成：
  1. 进入首页理解产品
  2. 搜索一个指数
  3. 查看一个模板对比摘要

### 6.2 工程验收

- `web` 可以稳定构建（`build:local` / `build:cloud` 无阻断）
- 核心接口无 5xx 阻断问题
- 页面无明显错位或空白状态失控

---

## 7. 发布门槛（Go / No-Go）

Go 条件：

1. In Scope 100% 完成
2. Out of Scope 未混入
3. 关键路径可演示（首页 → 指数页 → 模板对比）

No-Go 条件：

1. 首页缺失核心信息模块
2. 指数查询不可用
3. 模板对比摘要不可用

---

## 8. 版本后续衔接

- V2：补齐本地部署完整能力、数据深度、AI 实验功能
- V3：桌面打包、插件机制、社区分享闭环
