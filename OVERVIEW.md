# Anti-FOMO 项目总览

## 项目简介

Anti-FOMO 是一个智能化的股票组合波动监控工具，旨在帮助投资者：

1. **避免情绪化交易** - 通过理性的数据分析替代恐慌性决策
2. **及时发现风险** - 自动监控组合波动，超过阈值时主动提醒
3. **获得专业建议** - 可选的 AI 分析提供客观的投资建议
4. **节省时间** - 自动化日常监控，无需每天手动查看

## 核心功能

### ✅ 已实现

1. **组合配置管理** (`src/portfolio.py`)
   - 支持指数和个股
   - 灵活的资金配置（金额/比例）
   - 自动计算权重

2. **数据获取** (`src/data_fetcher.py`)
   - 集成 akshare，支持 A 股数据
   - 可扩展其他数据源

3. **波动率计算** (`src/volatility_calculator.py`)
   - 日收益率计算
   - 历史波动率（标准差）
   - 组合加权波动率

4. **智能阈值判断** (`src/threshold_manager.py`)
   - 双层阈值（警告/警报）
   - 组合级别 + 个股级别
   - 自动生成警报消息

5. **AI 分析** (`src/ai_analyzer.py`)
   - 接入 OpenAI GPT-4
   - 专业投资建议
   - 可自定义提示词

6. **多渠道通知** (`src/notification.py`)
   - 控制台输出
   - 文件日志
   - 邮件（预留接口）

7. **自动化调度** (`src/main.py`)
   - 定时任务
   - 单次运行模式
   - 可配置时间

## 技术栈

- **语言**: Python 3.7+
- **数据获取**: akshare
- **数据处理**: pandas, numpy
- **AI 分析**: OpenAI API
- **调度**: schedule
- **配置**: YAML

## 使用流程

```
1. 用户配置组合
   ↓
2. 程序获取最新数据 (akshare)
   ↓
3. 计算波动率
   ↓
4. 判断是否超过阈值
   ↓
5. 如果超过：调用 AI 分析
   ↓
6. 发送通知（控制台/文件/邮件）
   ↓
7. 记录日志
```

## 扩展性

### 可以轻松扩展的功能：

1. **新数据源**
   - 美股、港股、加密货币
   - 修改 `data_fetcher.py`

2. **新通知渠道**
   - 微信、钉钉、Telegram
   - 修改 `notification.py`

3. **更复杂的分析**
   - 技术指标（MACD、KDJ 等）
   - 风险价值（VaR）
   - 修改 `volatility_calculator.py`

4. **历史回测**
   - 分析历史波动
   - 优化阈值设置

## 使用建议

### 适合人群
- 长期投资者
- 指数基金定投者
- 容易受市场情绪影响的投资者

### 不适合人群
- 日内交易者（数据更新频率不足）
- 追求绝对收益的激进投资者

### 最佳实践

1. **首次使用**
   - 先设置较宽松的阈值（5%/8%）
   - 观察一段时间后调整

2. **定期回顾**
   - 每周查看一次日志
   - 分析哪些警报是真正需要关注的

3. **结合其他工具**
   - Anti-FOMO 专注于波动监控
   - 建议配合基本面分析使用

4. **AI 建议的使用**
   - AI 分析仅供参考
   - 最终决策需要结合个人情况

## 性能与成本

- **运行速度**: 单次检查约 10-30 秒（取决于持仓数量和网络速度）
- **资源占用**: 极低，可在树莓派等设备运行
- **API 成本**: 如启用 AI 分析，每次约 $0.01-0.05（GPT-4）

## 安全性

- 所有配置在本地
- 不上传任何持仓数据
- API Key 保存在本地配置文件
- 建议将 `config.yaml` 加入 `.gitignore`（已配置）
